## Como executar

### Requisitos

* Docker
* Docker Compose

### Passo a passo

* Instale os requisitos acima
* Clone esse repositório
* Na pasta raiz do repositório execute o seguinte comando

```
$ docker-compose up -d
```

### Portas

* 3000: API
* 5432: PostgreSQL
* 8080: pgAdmin4
